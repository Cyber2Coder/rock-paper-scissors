# 🪨📄✂️ Rock Paper Scissors

A browser-based Rock Paper Scissors game built as part of **The Odin Project Foundations** curriculum.  
This project demonstrates DOM manipulation, event-driven JavaScript, UI updates, and clean game logic.

## 📁 Project Structure
- `index.html` — UI layout and game interface  
- `style.css` — Styling, layout, and button states  
- `script.js` — Game logic, DOM updates, event listeners  

---

## 🎮 Features
- Player selects **Rock**, **Paper**, or **Scissors**  
- Computer generates a random choice  
- Round results displayed dynamically  
- Score tracking for both player and computer  
- Game ends when someone reaches **5 points**  
- Buttons disable after game completion  
- **Reset button** to start a new match  
- Clean, minimal UI

# Rock Paper Scissors — UI 3.0 (Epic Edition)

A joyful, animated, kid‑friendly browser game built with HTML, CSS, and JavaScript.  
Version 3 introduces a fully polished visual experience with floating elements, particle systems, and an epic victory screen.

## ✨ Features

### 🎨 UI 3.0 Visual Style
- Pastel sky gradient background with smooth animated flow  
- Floating shapes, clouds, hearts, stars, sparkles, and extra drifting elements  
- Toy‑style buttons with soft glow and hover animations  
- Animated scoreboard with win/lose feedback  

### 🦄 Unicorn Celebration
- Unicorn jump animation triggered after “Play Again”
- Rainbow trail particle system following the unicorn’s movement
- Confetti burst synced to victory

### 🌟 Epic End Screen
- Gold‑shine animated “YOU WIN!” title
- Starburst glow behind the message
- Power‑up style “Play Again” button
- Smooth fade‑in/out transitions

### 🧠 Game Logic
- Player vs Computer (randomized choice)
- Personality lines for each move
- Kid‑friendly win/lose/tie messages
- First to 5 wins triggers the end screen
- Double‑trigger prevention for stable game flow

## 🛠️ Tech Stack
- **HTML5** — structure
- **CSS3** — animations, gradients, effects
- **JavaScript** — game logic, particle systems, DOM manipulation

## 🚀 How to Play
1. Click Rock, Paper, or Scissors  
2. First to 5 points wins  
3. Enjoy the animations, effects, and unicorn celebration  
4. Click **Play Again** to restart
