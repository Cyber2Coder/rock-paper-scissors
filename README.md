🪨📄✂️ Rock Paper Scissors
A simple, interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript as part of The Odin Project Foundations curriculum. This version includes a clean UI, live score updates, and dynamic round results.

🚀 Features
Three selectable choices: Rock, Paper, Scissors

Randomized computer opponent

Live score tracking

Round‑by‑round result display

Clean, minimal UI

Fully functional in the browser — no dependencies
